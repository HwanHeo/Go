### 포인터 연산자
> `&`, `*`, `<-`
```Go
&: 현재 변수의 메모리 주소를 구합니다.
a := 1
b := &a // a의 메모리 주소를 b에 대입

*: 현재 포인터 변수의 저장된 메모리에 접근하여 값을 가져오거나 저장합니다.
a := new(int)
*a = 1  // a에 저장된 메모리에 접근하여 1을 저장

<-: 채널에 값을 보내거나 값을 가져옵니다.
c := make(chan int)
go func() {
    c <- 1   // 채널 c에 1을 보낸다.
}()
a := <-c   // 채널 c에서 값을 가져와서 a에 대입
```
